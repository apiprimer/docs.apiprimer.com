## Viewing Security Groups

1. Open the Amazon VPC console ([https://console.aws.amazon.com/vpc/]).
2. In the navigation pane, choose Security Groups.
3. The security groups for your account will be displayed in a list.
4. Click on the name of a security group to view its details.

![sgroups](~/assets/labs/sgroups.jpg)

## Viewing Network Access Control Lists (NACLs)

1. Open the Amazon VPC console ([https://console.aws.amazon.com/vpc/]).
2. In the navigation pane, choose VPCs.
3. Select the VPC `myvpc` for which you want to view NACLs.
4. Choose Subnets.
5. Select the subnet `mypublicsubnet` for which you want to view NACLs.
6. Choose the Network ACLs tab.
7. The NACLs that are associated with the selected subnet will be displayed in a list.
8. Click on the name of an NACL to view its details.

![nacl](~/assets/labs/nacl.jpg)
